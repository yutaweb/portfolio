───────────────────────────────────
     製品特長
───────────────────────────────────
「すぐ使えるCGI」は小規模サイト向けの CMS です。
複雑な機能はありませんが、簡単にサイトに導入でき、確実に動作します。
更新担当者のトレーニングは不要です。

商用利用も可能です。
プログラミングの知識が無くてもデザインを変更できますので、一旦導入の
コツをつかめばその後のデザイン変更や追加導入が非常に簡単です。

既存サイトやサイトの一部にも導入可能で、サイトの一部だけ CMS で
更新したいといった要望にも対応可能です。

《主な機能》
・PDFファイルなどをアップロードして掲載
・無制限数のファイル添付が可能
・拡張子を判別してアイコンを自動表示（PDF、エクセル、ワード、パワーポ
　イント、、ZIP、テキスト、画像、その他）
・日本語ファイル名に対応
・大きい画像を指定サイズまで縮小
・自動的に前後ページへのリンクが付きます
・指定件数で目次ページ分割
・URLとメールアドレスの自動リンク
・HTMLタグ入力可
・新着情報書き出し
・RSSファイル書き出し

───────────────────────────────────
     動作環境
───────────────────────────────────
・Perl による CGI の実行が可能な環境。（ Perl 5.6 - 5.14 ）
・必要な Perl モジュール： CGI.pm（標準モジュール）
・画像のリサイズを行う場合に必要： Image::Magick または GD モジュール
・UNIX系OS（CentOS、Linux、Red Hat、Fedora、FreeBSD、Solaris など）、
　Windows サーバ、Mac OS X で動作します。
・推奨Webサーバ： Apache、IIS（Internet Information Services）。
・管理画面の動作には、JavaScript が実行できるブラウザが必要です。
